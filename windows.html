Desktop Installation Guide

Download & Run Install

Get the latest .exe installer or .msi package from the official GitHub Releases and run the installation wizard.

System Tray Background Mode Tray

Once running, Fluence sits quietly in your system tray. Right-click the icon to adjust settings, view logs, or exit the application.

Global Shortcut Hotkeys

Press Ctrl + Shift + Space inside any application (such as Word, Discord, Chrome, or IDEs) to trigger dictation instantly.

Secure API Keys Security

Paste your Groq or OpenAI key in the settings. It is stored securely in the native Windows Credential Manager with zero cloud telemetry.

Overlay Indicator Visuals

A wave pill overlay appears on the bottom right of your screen during recording. The overlay is click-through to prevent focus loss.

Offline Engine Support Offline

Enable offline mode inside settings to download the SenseVoice model weights. Run local, private transcription with zero internet connectivity.
